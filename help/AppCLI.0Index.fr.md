Documentation de l'application Mynatime CLI
====================================

🇬🇧 [English version](AppCLI.0Index.en.md)

### Démarrage rapide

```bash
# 1. installer (Linux)
curl -fsSL https://raw.githubusercontent.com/sandrock/mynatime/refs/heads/master/packages/install.sh | bash

# 2. créer un profil
m pro add

# 3. récupérer les catégories d'activité depuis le service
m act cat refresh

# 4. commencer le suivi
m act start monprojet
m act stop

# 5. réviser et publier
m status
m commit
```
